# AXUR CHALLENGE

This is a test for a Front-End developer position at Axur.

## Run the project
1. git clone https://github.com/JohnOliver23/challenge-axur.git
2. cd challenge-axur/
3. npm install
4. npm start



## Technologies:
- Typescript
- HTML
- CSS

## Framework
### React FC with hooks
- useState for state control
- useEffect for lifecicle control

## Libraries
- Styled components
- Formik
- Yup
- React Router DOM
- React icons
